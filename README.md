# Low Battery Pet

## 產品概述
Low Battery Pet 是一個以行動裝置優先的情感陪伴 Web 應用。
產品核心聚焦在輕量的每日 check-in 與可持續的寵物心情循環。

## 目前階段
第 1 階段基線：僅建立文件。
此儲存庫目前先建立範圍、規則與方向，暫不進入應用實作。

## 規劃技術棧
- React
- Vite
- TypeScript
- styled-components
- Vitest
- Testing Library
- npm

## MVP 範圍（規劃）
- 以行動優先為主的基礎應用外殼
- 簡單的 check-in 互動流程
- 由使用者輸入驅動的寵物心情狀態更新
- 最小化本地行為（MVP 基線規劃中不依賴後端）

## 不在範圍內（目前階段）
- UI 畫面實作
- 寵物狀態引擎實作
- 後端或 Supabase 整合
- AI API 整合
- 身分驗證
- PWA 支援
- Tailwind CSS 或外部 UI 框架
- Redux、Zustand 或 React Query

## 儲存庫規則（摘要）
- AI / 程式開發流程規則以 `AGENTS.md` 為準。
- 產品與架構方向以 `DESIGN.md` 為準。
- 變更需維持小範圍且以單一 Issue 為界。
